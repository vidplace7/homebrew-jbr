# Unofficial Jetbrains Runtime Tap
🍺 Homebrew Tap for Jetbrains Runtime

Supported: 🍎 MacOS

## Packages

- [jbr](Casks/jbr.rb): JBR Latest (25)
- [jbr@21](Casks/jbr@21.rb): JBR 21
- [jbr@17](Casks/jbr@17.rb): JBR 17

## Installation

`brew install vidplace7/jbr/jbr@21`

Or
```shell
brew tap vidplace7/jbr
brew install jbr@21
```

## Documentation

- `brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
